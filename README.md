# Drum-Kit
Play with the Drums

## How to use

Go <a href="https://2x-hra.github.io/Drum-Kit/">here</a> (wait like 10 seconds to load the sounds). 

As you see there are 7 types of drum tools with a different type of key to press:
- Tom-1 sound : `w` key
- Tom-2 sound : `a` key
- Tom-3 sound : `s` key
- Tom-4 sound : `d` key
- Snare sound : `j` key
- Crash sound : `k` key
- Kick-Bass sound : `l` key

and HAVE FUN !
